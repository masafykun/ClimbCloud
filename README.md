# ClimbCloud

> Unity 2D アクションゲーム — 雲を登っていくネコのアクションゲーム

## 概要

Unity で制作した 2D 横スクロール・アクションゲームです。
ネコのキャラクターが雲を登りながらゴールを目指します。

- スペースキーでジャンプ、左右矢印キーで移動
- アニメーション（歩行・ジャンプ）つきのキャラクター制御
- ゴール地点に触れるとクリア画面に遷移
- 雲などのステージオブジェクトはプレハブで管理

## 使用技術

- Unity（C#）
- Unity 2D Physics（Rigidbody2D）
- Unity Animation（Animator）

## 動作環境 / 注意事項

- Unity でプロジェクトを開いて実行してください
- 動作確認済みの Unity バージョンは不明です
